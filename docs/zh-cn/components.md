# 组件概览

## 核心视图

- `WebCutEditor`：编辑器主界面
- `WebCutPlayer`、`WebCutPlayerScreen`、`WebCutPlayerButton`：播放器视图
- `WebCutManager`、`WebCutManagerScaler`：时间线管理与缩放
- `WebCutLibrary`：素材库（图片、视频、音频、文本）
- `WebCutProvider`：上下文提供者

## 片段视图

- `WebCutVideoSegment`、`WebCutAudioSegment`、`WebCutImageSegment`、`WebCutTextSegment`

## 工具视图

- `WebCutSplitTool`、`WebCutSplitKeepLeftTool`、`WebCutSplitKeepRightTool`
- `WebCutDeleteTool`、`WebCutClearTool`
- `WebCutSelectAspectRatio`、`WebCutPanel`、`WebCutBasicPanel`、`WebCutTextPanel`

## 基础组件

- `AdjustableBox`、`AudioShape`、`ContextMenu`、`ScrollBox`、`DraggableHandler`、`RotateInput`
